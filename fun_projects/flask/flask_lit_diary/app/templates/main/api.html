API Overview

The Literature Diary API provides programmatic access to public notes and allows authenticated users to create, update, and delete their own content.

Base URL

/api/v1

Authentication

  • Public: read-only access to notes
  • Authenticated: create, update, and delete notes
  • Session-based authentication using Flask-Login

Notes Endpoints

GET /api/v1/notes

Retrieve all public notes.

POST /api/v1/notes

Create a new note (authentication required).

PUT /api/v1/notes/<id>

Update an existing note (owner only).

DELETE /api/v1/notes/<id>

Delete a note (owner only).

id — integer note identifier

Full API description and examples are available on GitHub.

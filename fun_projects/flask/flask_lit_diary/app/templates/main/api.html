Overview

This project provides a REST API that allows programmatic access to notes and related resources.

Base URL

/api/v1

Authentication

  • Read operations are public
  • Write operations require authentication
  • Session-based authentication (Flask-Login)

Notes Endpoints

Get all notes

GET /api/v1/notes

Create a note

POST /api/v1/notes

Update a note

PUT /api/v1/notes/<id>

Delete a note

DELETE /api/v1/notes/<id>

id — integer note identifier

Full API description and examples are available on GitHub.
